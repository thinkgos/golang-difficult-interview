# Pod Probe

## 一. Init 容器

```yaml
apiVersion: v1
kind: Pod
metadata:
  name: init-demo
spec:
  initContainers:
    - name: install
      image: busybox:1.32.0
      command:
        - touch
        - /work-dir/index.html
      volumeMounts:
        - name: workdir
          mountPath: "/work-dir"
  containers:
    - name: init-container
      image: thinkgos/helloworld:v1
      ports:
        - containerPort: 80
      readinessProbe:
        exec:
          command:
            - cat
            - /ready/index.html
      volumeMounts:
        - name: workdir
          mountPath: /ready
  volumes:
    - name: workdir
      emptyDir: {}
```

## 二. 就绪检测

readinessProbe-httpget

```yaml
apiVersion: v1
kind: Pod
metadata:
  name: readiness-pod
spec:
  containers:
    - name: readiness-container
      image: thinkgos/helloworld:v1
      imagePullPolicy: IfNotPresent
      readinessProbe:
        exec:
          command:
            - cat
            - /usr/share/nginx/html/index.html
        initialDelaySeconds: 5
        timeoutSeconds: 1
    - name: file-ready
      image: busybox:1.32.0
      command: ["/bin/sh","-c","sleep 5; mkdir -p /usr/share/nginx/html; touch /usr/share/nginx/html/index.html; sleep 3600"]
```

## 三. 存活检测- liveness probe

### a. livenessProbe-exec

```yaml
apiVersion: v1
kind: Pod
metadata:
  name: liveness-exec-pod
spec:
  containers:
    - name: liveness-exec-container
      image: busybox:1.32.0
      imagePullPolicy: IfNotPresent
      command: ["/bin/sh","-c","touch /tmp/live ; sleep 60; rm -rf /tmp/live; sleep 3600"]
      livenessProbe:
        exec:
          command: ["test","-e","/tmp/live"]
        initialDelaySeconds: 1
        periodSeconds: 3
```

### b. livenessProbe-httpget

```yaml
apiVersion: v1
kind: Pod
metadata:
  name: liveness-httpget-pod
  namespace: default
spec:
  containers:
    - name: liveness-httpget-container
      image: thinkgos/helloworld:v1
      imagePullPolicy: IfNotPresent
      ports:
        - name: http
          containerPort: 80
      livenessProbe:
        httpGet:
          port: http
          path: /index.html
        initialDelaySeconds: 1
        periodSeconds: 3
        timeoutSeconds: 10
```

### c. livenessProbe-tcp

```yaml
apiVersion: v1
```

## 四. lifecycle - postStart,preStop

```yaml
apiVersion: v1
kind: Pod
metadata:
  name: lifecycle-demo
spec:
  containers:
    - name: lifecycle-demo-container
      image: thinkgos/helloworld:v1
      lifecycle:
        postStart:
          exec:
            command: ["/bin/sh", "-c", "echo Hello from the postStart handler > /usr/share/message"]
        preStop:
          exec:
            command: ["/bin/sh", "-c", "echo Hello from the poststop handler > /usr/share/message"]
```
